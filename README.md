# Taverna-v2

Minimal Orchestration Bridge for SillyTavern.

This project enables remote control of SillyTavern via external agents (like Antigravity or n8n) using natural language or STscripts.

## Project Goal
**Antigravity / VS Code** -> **Taverna-v2** -> **SillyTavern** -> **Control Total**.

## Structure
- `/orchestrator`: Core bridge logic (Backend Plugin & Frontend Extension).
- `/scripts`: Installation and diagnostic tools.
- `/docs`: Functional and architectural specs.
- `/bridge`: (Coming soon) High-level wrappers for Antigravity.
